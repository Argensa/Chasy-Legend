READ ME
-------
Asset: Voxel Buildings 
Version: 1.0
Date: 30/09/2019 

- This asset is based on Voxel Art.
- This asset includes flat color palette Texture.
- Textures should be used with shader<<mobile<<diffuse for actual results.
- The package includes two demo scenes: 
1. Daylight: Directional light is used. 
2. Night View: Spot lights & point lights are used for Building lights. Directional light with 0.4 intensity is used to create night view. 
- No Skybox is used.
- Color Light is used. Please check Environment Lighting for details. 
- Post Processing Effects used: 
1. Bloom Effect
2. Color grading
